## 📚 스터디 규칙

### 문제 풀이
- 매주 **5문제**씩 풀이
- 매주 **월요일**에 새로운 문제 업데이트
- 문제 풀이 기한: **일요일 23:59**까지

### 코드 제출 규칙
- 각자의 브랜치에서 정답 코드 업로드 후, 문제 풀이 기한에 맞춰 Pull Request를 올립니다.
(파일명: `[BOJ-문제번호] 문제제목.확장자`, 아래 **_폴더 구조_** 참고)
- 주차가 지날 때마다, 서로의 PR을 검토 후 merge합니다.

### 코드 리뷰
- 매 주마다 다른 사람의 코드에 대한 피드백 / 느낀점 제공
